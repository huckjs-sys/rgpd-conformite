# RGPD Compliance — ChurchCRM Community Plugin

GDPR / RGPD compliance toolkit for ChurchCRM. Helps churches based in the EU (and wider GDPR-aligned jurisdictions) meet their obligations under articles 12-22 of the GDPR.

- **Plugin ID:** `rgpd-conformite`
- **Type:** Community
- **Author:** Eoles Conseil
- **Minimum ChurchCRM version:** 7.0.0
- **License:** MIT

## Planned features (roadmap)

| Feature | GDPR article | Status |
|---|---|---|
| Data Subject Access Request (DSAR) — export a person's data as JSON / PDF | Art. 15 (right of access) | planned |
| Right to erasure — anonymise or delete a person on request | Art. 17 (right to be forgotten) | planned |
| Processing register — admin form to declare which categories of data are processed and why | Art. 30 | planned |
| Retention policy — automatic anonymisation of inactive records after configurable period | Art. 5(1)(e) | planned |
| Consent log — track explicit consents per person (newsletter, photo, etc.) | Art. 7 | planned |
| Data breach notification template | Art. 33 | planned |
| DPO contact details in plugin settings | Art. 37 | planned |

## Installation

This plugin is a community plugin for ChurchCRM. It will be installable via the URL installer once approved.

1. Open Admin → Plugins
2. Click "Install from URL"
3. Paste the release URL
4. Confirm the SHA-256
